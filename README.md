# Three Buttons React App

This is a simple React application with three buttons. When you click a button, a message appears on the screen.

## Features

- Three buttons
- Click event handling
- useState hook
- Centered layout

## Technologies

- React.js
- JavaScript
- Vite

## Installation

1. Clone the repository:

```bash
git clone [https://github.com/romanbetsapp2023-sudo/project_for_up.git](https://github.com/romanbetsapp2023-sudo/project_for_up.git)
```

2. Go to the project folder:

Bash
cd project_for_up 3. Install dependencies:

Bash
npm install 4. Start the project:

Bash
npm run dev
The app will open at:

http://localhost:5173/
